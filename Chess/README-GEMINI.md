# Google Gemini API Integration Setup

## How to Get Your API Key

1. **Go to Google AI Studio**: https://makersuite.google.com/app/apikey
2. **Sign in** with your Google account
3. **Create a new API key** by clicking "Create API Key"
4. **Copy the API key** (it looks like: `AIzaSyC...`)

## How to Use

1. **First time**: Click the "🤖 AI Phân Tích" button
2. **Enter your API key** when prompted
3. **The key will be saved** in your browser for future use
4. **Click the button again** to get AI analysis of the current position

## Features

- **Board Analysis**: AI analyzes the current board position
- **Move Suggestions**: Suggests the best move for your Rook
- **Tactical Explanation**: Explains why the move is good
- **Strategic Notes**: Provides tactical considerations

## Free Tier Limits

- **15 requests per minute**
- **1500 requests per day**
- **No credit card required**

## How It Works

1. The AI receives the current board state
2. It analyzes the position using Xiangqi knowledge
3. It suggests the best move with explanation
4. Results are displayed in the move list

## Troubleshooting

- **"API key required"**: Enter your API key when prompted
- **"API call failed"**: Check your internet connection
- **"Analysis failed"**: Try again (API might be busy)

## Privacy

- Your API key is stored locally in your browser
- Board positions are sent to Google for analysis
- No personal data is collected
